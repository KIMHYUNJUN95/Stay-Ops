# Next Meeting Agenda

## Goal

Decide the first product direction clearly enough to start detailed planning.

## Agenda

### 1. Product Identity

- Is `StayOps` the final name or a working name?
- Should the product target hotels only, or all accommodation businesses?

### 2. First Customer

- Who is the first target user?
- Small hotel?
- Business hotel?
- Ryokan?
- Motel?
- Pension?
- Guesthouse?

### 3. MVP Scope

Choose the first 3 must-have modules:

- Lost and found
- Maintenance
- Order requests
- Housekeeping
- Handover notes
- Notifications
- Admin dashboard

### 4. Platform

Choose first launch target:

- iOS only
- Android only
- iOS and Android
- Mobile app plus admin web later

### 5. Technical Stack

Compare:

- React Native + Expo
- Flutter
- Supabase
- Firebase

### 6. Design Direction

Decide:

- Simple and utilitarian
- Premium hotel operations
- Japanese business app style
- Global SaaS style

### 7. Free Plan

Define free-start limits:

- Number of hotels
- Number of staff
- Monthly task count
- Photo storage limit
- Notification limit
